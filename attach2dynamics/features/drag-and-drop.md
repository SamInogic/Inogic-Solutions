# Drag & Drop

Users can drag and drop multiple files and folders at one time to SharePoint, Azure Blob Storage or Dropbox from Dynamics 365 CRM.&#x20;

{% hint style="info" %}
**Note: Drag & Drop of files and folders is not supported in IE.**
{% endhint %}
